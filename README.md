Biblioteca  que interage com webservice de Nota Fiscal de Serviços Eletrônica (NFS-e).

## Cidades Atendidas

* Curitiba

## Instalação 

```console
git clone https://github.com/marcelobelli/PyNFSe.git
cd PyNFSe
pip install -r requirements.txt
```
